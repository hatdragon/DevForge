# Event Monitor

The Event Monitor allows you to observe World of Warcraft events as they fire, along with their payloads.

It helps answer:

**“What events are firing, and what data do they carry?”**

---

## Overview

The Event Monitor provides a live stream of events dispatched by the WoW client.

- Real-time event display
- Payload inspection
- Filtering and categorization
- Pause and resume controls

## Capabilities

### Live Monitoring
- Observe events as they occur
- Inspect argument values in order
- Expand table payloads safely

### Filtering
- Filter by event name
- Group events by category
- Temporarily pause noisy streams

### Exploration
- Browse known events
- Discover undocumented or rarely-used events

---

## When to Use the Event Monitor

Use the Event Monitor when:
- Debugging event-driven behavior
- Determining which events drive UI updates
- Inspecting payload formats
- Investigating unexpected behavior
