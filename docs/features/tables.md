# Table Viewer

The Table Viewer allows inspection and exploration of Lua tables at runtime.

It is designed to make large or deeply nested tables understandable.

---

## Overview

Lua tables are the backbone of addon state, configuration, and runtime behavior.

The Table Viewer allows you to:
- Explore table structure interactively
- Expand nested tables
- Inspect values safely
- Dump or copy table contents

## Capabilities

- Browse globals, addon data, and namespaces
- Expand nested tables incrementally
- View key/value pairs clearly
- Dump or copy table content for analysis

---

## When to Use the Table Viewer

Use the Table Viewer when:
- Inspecting SavedVariables
- Debugging runtime state
- Exploring unknown data structures
- Verifying assumptions about table contents
